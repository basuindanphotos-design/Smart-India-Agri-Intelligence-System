About the Project

AI-powered crop price prediction system built for Indian agriculture markets.

What is crop price prediction?

Crop price prediction estimates upcoming mandi modal prices using historical market records and contextual attributes such as state, district, market, commodity, variety and arrival date.

Business value for farmers and traders

The platform helps users compare expected prices with current market behavior, identify stronger markets, and improve selling or procurement timing with data-backed confidence.

How this project is structured

  • Dataset explorer to inspect the exact training dataset rows.
  • Dashboard with trend, monthly, state and commodity visual analytics.
  • Prediction module with connected State, District, Market, Commodity and Variety workflow.
  • Model evaluation module for comparative train/test metrics.
  • Data preprocessing and EDA pages for transparency of pipeline steps.

Dataset source and schema

The training file is `dataset/crop_prices.csv` and follows AGMARKNET-style mandi records with key fields: State, District, Market, Commodity, Variety, Arrival Date, Min Price, Max Price and Modal Price.

Technology stack

  • Frontend: HTML5, CSS3, Bootstrap 5, JavaScript, Chart.js
  • Backend API: Python Flask
  • ML pipeline: scikit-learn with encoded categorical features
  • Data processing: Pandas and NumPy
  • Model persistence: Joblib

Real-time data behavior

Dashboard and home metrics are loaded from backend APIs and refreshed periodically, so uploaded datasets and retrained models are reflected directly in the UI.
