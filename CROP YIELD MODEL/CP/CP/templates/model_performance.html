Algorithm Notes

  • CatBoost is a gradient boosted decision tree library optimised for categorical features without one-hot encoding overhead.
  • It uses ordered boosting to prevent prediction shift and target leakage during training.
  • The model handles missing values natively without imputation, making it robust on sparse agronomic datasets.
  • Feature importance is computed using SHAP-based scores internally, giving reliable attribution per predictor.
  • Regularisation parameters (L2 leaf reg, depth, iterations) were tuned via cross-validation to avoid overfitting on the India crop dataset.
