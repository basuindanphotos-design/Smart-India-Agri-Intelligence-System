Project Workflow

End-to-end machine learning pipeline from raw data to deployment-ready prediction service.

Dataset Collection

Collected crop, rainfall, pesticide, temperature, and yield records across countries and years to capture broad agricultural patterns.

Data Preprocessing

Cleaned missing values, standardized schema, and prepared categorical and numerical fields for model-ready transformation.

Exploratory Data Analysis

Identified variable distributions, outliers, and interactions among rainfall, temperature, pesticide usage, and crop output.

Feature Engineering

Selected predictive features and structured them for preprocessing pipelines to ensure consistent training and inference behavior.

Model Training

Trained regression model using transformed features and validated generalization performance on held-out samples.

Model Evaluation

Measured R2, MAE, and RMSE and compared against baseline to validate predictive value beyond naive estimators.

Prediction System

Integrated model and preprocessor into Flask API endpoints for real-time crop yield prediction and interpretation.

Web Deployment

Delivered a responsive multi-page analytics dashboard for demonstrations and decision support across devices.
